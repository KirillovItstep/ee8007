Использование программы для создания раскладки шкал и простановки номеров.

В главном окне задается начальный номер и количество шкал. Также вводятся начальные координаты первой шкалы, расстояние между шкалами по горизонтали и вертикали, ширина и высота раскладки.

При нажатии на кнопку «ОК» появляется диалоговое окно открытия файла одиночной шкалы CorelDraw. Файл одиночной шкалы должен иметь невидимую прямоугольную рамку на всех слоях, так как это нужно для правильного определения ширины и высоты изображений на каждом слое. Все слои одиночной шкалы должны быть заданы как видимые.

В файлах настроек с расширением “.ini”, а по умолчанию это файл “settings.ini”, задаются смещения шкал, если положение отдельных шкал требует коррекции. Также задаются названия слоев для копирования. Название первого по порядку слоя определяет, на каком слое будет размещен номер.

В том же файле задается шрифт, высота, цвет надписи для номера. Количество знаков для номера определяется по исходному количеству знаков, заданных в поле "number". Координаты положения номера также задаются.

Если потребуется перейти на другую раскладку, нужно выбрать в выпадающем списке имя файла с сохраненными настройками. Файл настроек можно скопировать, используя средства Windows. Сохранение настроек происходит автоматически после нажатия на кнопку «ОК».

Номер ставится на слое, который следует первым в файле настроек.
